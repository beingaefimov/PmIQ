# PM Process Tailoring

Этот скилл реализует принцип PMBOK 8 "Адаптация" (Tailoring). Он помогает аналитикам PMO и руководителям проектов создавать кастомные решения без глубокого программирования.

## Когда использовать
- Когда пользователю нужно создать специализированного помощника (например, "Агент по контролю контрактов" или "Scrum-мастер для конкретного проекта").
- Когда стандартные процессы компании слишком тяжелы для малого проекта и их нужно упростить.

## Инструкции для агента
1. Оцени контекст и сложность проекта (размер команды, критичность, уровень неопределенности).
2. Предложи использовать `pm-declarative-agent-developer` для генерации JSON-манифеста нового агента, специфичного для этой задачи.
3. Если нужно разработать кастомную программу, предложи `pm-pc-builder` для создания программы.
4. Предоставь пользователю пошаговый план (scaffolding) создания и развертывания этого кастомного решения.

## Используемые MCP инструменты
- `scaffold_declarative_agent`
- `assess_project_complexity`
