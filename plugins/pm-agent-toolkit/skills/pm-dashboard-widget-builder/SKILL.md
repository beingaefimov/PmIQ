# PM Dashboard Widget Builder

Рендеринг сложных данных в UI чата.

## Инструкции для агента
1. Когда пользователю нужны визуальные данные (например, гистограмма ресурсов или тепловая карта рисков), используй `generate_widget_schema`.
2. Передавай в инструмент только структурированные данные, полученные от других MCP-инструментов. Не пытайся генерировать схему виджета "из головы", чтобы избежать синтаксических ошибок JSON.
