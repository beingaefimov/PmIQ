# PM PC Builder

Выполнение кастомных программ.

## Инструкции для агента
1. Когда пользователю нужно запустить программу, скрипт, следуй инструкциям здания программ.
2. Передавай в инструмент только структурированные данные, полученные от других MCP-инструментов. Не пытайся генерировать программы "из головы", чтобы избежать синтаксических ошибок и ошибок JSON.
