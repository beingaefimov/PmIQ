# PM Declarative Agent Developer

Инструмент для создания кастомных агентов без написания кода.

## Инструкции для агента
1. Используй `scaffold_declarative_agent` для генерации базовой структуры JSON-манифеста.
2. Помоги пользователю определить инструкции (instructions), знания (knowledge sources) и действия (actions) для нового агента.
3. Предложи проверить манифест на соответствие корпоративным стандартам безопасности.
