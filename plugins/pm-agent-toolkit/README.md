# PM Agent Toolkit

Toolkit for PMO analysts to build custom declarative agents, dashboard widgets, and tailor PM processes to project complexity.

## Installation
```bash
copilot plugin install ./plugins/pm-agent-toolkit
```

## Available Skills
| Skill Name | Description |
|------------|-------------|
| `pm-process-tailoring` | Helps adapt PM processes and templates to specific project complexity (Tailoring). |
| `pm-declarative-agent-developer` | Scaffolds JSON manifests for creating custom Copilot agents (e.g., "Compliance Agent"). |
| `pm-dashboard-widget-builder` | Generates OpenAI Apps SDK widget schemas for rendering charts in Copilot Chat. |

## Usage Examples
- "Help me create a declarative agent that acts as a Risk Officer for my project."
- "Generate a widget schema to display a resource histogram in the chat."
- "How should I tailor the PM processes for a small, low-complexity internal project?"
