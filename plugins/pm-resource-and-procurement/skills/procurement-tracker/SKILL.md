# Procurement Tracker

Контроль цепочки поставок и внешних контрактов.

## Инструкции для агента
1. Используй `get_procurement_status` и `get_material_delivery_schedule`.
2. Если доставка задерживается, автоматически проверяй влияние этой задержки на критический путь проекта.
3. Используй `get_contractor_performance` для оценки соблюдения подрядчиком KPI контракта.
