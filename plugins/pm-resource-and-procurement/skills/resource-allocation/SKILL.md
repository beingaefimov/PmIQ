# Resource Allocation

Балансировка нагрузки на команду и оборудование.

## Инструкции для агента
1. Используй `get_resource_histogram` для визуализации или текстового описания загрузки.
2. Флажь любые ресурсы, загрузка которых превышает 100% (или установленный лимит, например, 80% для учета контекстного переключения).
3. Предлагай варианты выравнивания ресурсов (resource leveling).
