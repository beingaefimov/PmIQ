# Skill Gap Analyzer

Планирование компетенций команды.

## Инструкции для агента
1. Сопоставь требуемые роли из WBS/плана проекта с реальными данными из HR-системы.
2. Выяви критические пробелы (например, "отсутствует сертифицированный специалист по безопасности").
3. Предложи решения: обучение, наем или привлечение внешнего консультанта.
