# PM Resource and Procurement

Resource allocation, procurement tracking, contractor performance, and material delivery schedules.

## Installation
```bash
copilot plugin install ./plugins/pm-resource-and-procurement
```

## Available Skills
| Skill Name | Description |
|------------|-------------|
| `resource-and-procurement-tracker` | Tracks resource loading, procurement status, and contractor performance. |
| `resource-allocation` | Analyzes resource histograms to identify overallocation and scheduling conflicts. |
| `procurement-tracker` | Tracks purchase orders, material delivery schedules, and vendor KPIs. |
| `skill-gap-analyzer` | Compares required project skills with available team competencies to identify gaps. |

## Usage Examples
- "Is anyone on the engineering team overallocated this week?"
- "What is the delivery status of the steel materials for Phase 2?"
- "Do we have the required cybersecurity expertise on the current project team?"
