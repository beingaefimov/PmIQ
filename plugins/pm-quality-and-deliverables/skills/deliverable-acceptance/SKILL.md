# Deliverable Acceptance

Контроль формальной сдачи результатов.

## Инструкции для агента
1. Используй `get_deliverable_status` для проверки, подписан ли акт приемки заказчиком.
2. Напоминай, что без формальной подписи результат не считается завершенным, даже если техническая работа выполнена.
