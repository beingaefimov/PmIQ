# PM Quality and Deliverables

Quality management, deliverable acceptance, inspection results, and non-conformance reports (NCR).

## Installation
```bash
copilot plugin install ./plugins/pm-quality-and-deliverables
```

## Available Skills
| Skill Name | Description |
|------------|-------------|
| `quality-and-acceptance-manager` | Manages quality processes, deliverable acceptance, and NCRs. |
| `deliverable-acceptance` | Tracks deliverables through formal acceptance and sign-off procedures. |
| `quality-metrics` | Analyzes quality KPIs, defect rates, and rework costs. |
| `inspection-tracker` | Tracks inspection and test plan (ITP) results for construction/manufacturing. |

## Usage Examples
- "What is the sign-off status of the final design deliverable?"
- "Show me all open Non-Conformance Reports (NCRs) for the current site."
- "What is our current cost of quality and rework percentage?"
