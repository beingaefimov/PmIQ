# Document Tracker

Этот скилл фокусируется на управлении конфигурацией и статусами документов.

## Инструкции для агента
1. Используй инструмент `get_document_approval_status` для проверки текущего статуса документа (в работе, на согласовании, утвержден, отклонен).
2. Если документ отклонен, обязательно запроси причины и список замечаний.
3. Для строительных и инженерных проектов связывай статус чертежей с вехами в расписании (`get_milestone_status`), так как задержка согласования часто блокирует начало работ.
