# PM Risk and Change

Formal risk registers, change control (CCB), issue tracking, and contingency reserve management.

## Installation
```bash
copilot plugin install ./plugins/pm-risk-and-change
```

## Available Skills
| Skill Name | Description |
|------------|-------------|
| `risk-and-change-controller` | Manages formal risk registers, CCB change requests, and contingency reserves. |
| `risk-register-query` | Queries formal risk registers with probability, impact, and mitigation plans. |
| `change-request-tracker` | Tracks the status of change requests through the Change Control Board (CCB). |
| `issue-escalation` | Identifies overdue issues and suggests escalation paths for rapid resolution. |

## Usage Examples
- "Show me the top 3 high-impact risks and their current mitigation status."
- "What is the approval status of CR-2024-015?"
- "List all blocked issues that have exceeded their SLA response time."
