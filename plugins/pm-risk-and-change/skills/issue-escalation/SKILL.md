# Issue Escalation

Управление уже реализовавшимися проблемами.

## Инструкции для агента
1. Используй `get_issue_log` для поиска проблем со статусом "Open" или "Blocked", превышающих SLA по времени реакции.
2. Предлагай конкретные шаги эскалации на основе матрицы эскалации проекта (указанной в уставе проекта).
