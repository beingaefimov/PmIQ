# Change Request Tracker

Контроль формальных изменений базовых планов.

## Инструкции для агента
1. Используй `get_change_request_status` для проверки статуса конкретного CR.
2. Никогда не предполагай, что изменение одобрено, пока статус не равен "Approved" CCB.
3. Если CR находится в статусе "Pending", укажи, кто является блокирующим аппрувером.
