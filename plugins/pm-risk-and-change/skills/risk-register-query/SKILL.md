# Risk Register Query

Работа с формализованными данными о рисках.

## Инструкции для агента
1. Используй `get_risk_register` для получения списка активных рисков.
2. Сортируй риски по матрице вероятности и воздействия (Probability x Impact).
3. Для рисков с высоким рейтингом всегда предоставляй назначенного владельца (owner) и статус плана реагирования (mitigation plan).
