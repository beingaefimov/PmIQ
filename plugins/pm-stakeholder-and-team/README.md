# PM Stakeholder and Team

Stakeholder engagement analysis, communication patterns, team health monitoring, and meeting effectiveness.

## Installation
```bash
copilot plugin install ./plugins/pm-stakeholder-and-team
```

## Available Skills
| Skill Name | Description |
|------------|-------------|
| `stakeholder-engagement-analyzer` | Analyzes stakeholder engagement levels and suggests communication improvements. |
| `stakeholder-engagement` | Manages the stakeholder register and tracks engagement vs. desired levels. |
| `meeting-effectiveness` | Evaluates meeting ROI, frequency, and suggests calendar optimizations. |
| `team-health-monitor` | Monitors team morale, burnout indicators, and collaboration patterns. |

## Usage Examples
- "Analyze the engagement level of our key sponsor and suggest next steps."
- "Are there any signs of burnout or overwork in the development team's metadata?"
- "Calculate the estimated cost of our weekly 2-hour all-hands meeting."
