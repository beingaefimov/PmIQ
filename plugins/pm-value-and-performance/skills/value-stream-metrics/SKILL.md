# Value Stream Metrics

Фокус на домене "Измерение" для гибких методологий.

## Инструкции для агента
1. Используй `analyze_kpi_trends` для получения данных о времени цикла (cycle time) и пропускной способности (throughput).
2. Выявляй узкие места (bottlenecks): если cycle time растет, а throughput падает, укажи на это как на риск.
3. Сравнивай текущие метрики с историческими базовыми значениями проекта.
