# EVM and Value Analyzer

Этот скилл фокусируется на домене производительности "Измерение" (Measurement) из PMBOK 8, оценивая не просто затраты, а реальную доставку ценности.

## Когда использовать
- При запросах о финансовом здоровье проекта, отклонениях от бюджета или графика.
- Когда нужно оценить, оправдывают ли текущие затраты получаемую бизнес-ценность.

## Инструкции для агента
1. Запроси финансовые и временные данные через `get_budget_vs_actual` и `calculate_evm`.
2. Рассчитай и интерпретируй ключевые метрики:
   - CPI (Cost Performance Index): < 1.0 означает перерасход.
   - SPI (Schedule Performance Index): < 1.0 означает отставание от графика.
3. Не просто выдавай цифры. Объясни их бизнес-смысл: "Проект имеет CPI 0.85, что означает перерасход 15%. При текущих темпах прогнозируемая стоимость завершения (EAC) превысит бюджет".
4. Сопоставь эти данные со стратегическими OKR, используя `get_okr_alignment`, чтобы показать, влияет ли перерасход на конечную ценность для бизнеса.

## Используемые MCP инструменты
- `calculate_evm`
- `get_budget_vs_actual`
- `analyze_kpi_trends`
- `get_okr_alignment`
