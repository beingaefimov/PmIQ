# PM Value and Performance

Value delivery, EVM (Earned Value Management), KPIs, and OKR alignment for measuring project performance.

## Installation
```bash
copilot plugin install ./plugins/pm-value-and-performance
```

## Available Skills
| Skill Name | Description |
|------------|-------------|
| `evm-and-value-analyzer` | Analyzes project efficiency using EVM (CPI, SPI) and business value delivery. |
| `value-stream-metrics` | Analyzes flow metrics (cycle time, lead time, throughput) for adaptive projects. |
| `okr-progress-tracker` | Tracks the alignment of project deliverables with strategic company OKRs. |

## Usage Examples
- "Calculate the CPI and SPI for Project Alpha and explain the variance."
- "How is our cycle time trending compared to last quarter?"
- "Which OKRs is the current mobile app project contributing to?"
