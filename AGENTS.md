# AGENTS.md - Context for AI Assistants

Цель: Главный контекстный документ для AI-агентов (например, GitHub Copilot), которые работают внутри этого репозитория. Он объясняет ИИ, как устроен проект и какие правила нужно соблюдать при внесении изменений.
План содержимого:

## 1. Project Overview
- [ ] Краткое описание: PM IQ — это маркетплейс MCP-плагинов для управления проектами (PMBOK 8).
- [ ] Основная цель: Предоставить AI-агентам доступ к корпоративным данным (Jira, SAP, Primavera) через стандартизированные навыки (Skills) и инструменты (Tools).

## 2. Repository Architecture
- [ ] Описание структуры папок (`plugins/`, `servers/`, `samples/`, `scripts/`).
- [ ] Объяснение связи: Плагин -> `.mcp.json` -> `skills/` -> `SKILL.md` + `.csv`.
- [ ] Роль `server.json` и `.github/plugin/marketplace.json`.

## 3. Strict Rules for Modifying the Repo
- [ ] **Правило 1:** При добавлении нового скилла его `description` в YAML frontmatter НЕ ДОЛЖЕН превышать 1024 символа (иначе Copilot CLI его проигнорирует).
- [ ] **Правило 2:** При создании нового плагина или скилла ОБЯЗАТЕЛЬНО обновить `.github/plugin/marketplace.json`.
- [ ] **Правило 3:** Имена CSV-файлов с мок-данными должны в точности совпадать с именем инструмента (tool) из `.mcp.json`.
- [ ] **Правило 4:** Всегда запускать `python scripts/validate_marketplace.py` перед фиксацией изменений.
